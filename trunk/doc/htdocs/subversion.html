Accessing the subversion repository

You can check out the unstable branch (trunk) from the subversion repository with:

svn co https://svn.sourceforge.net/svnroot/sagaengine/trunk sagaengine

If you are using windows, take a look at TortoiseSVN. Use the same repository URL as above.

You can browse the subversion repository through the web interface as well. But browsing through the API docs which are generated from trunk every night is probably more worthwhile.
